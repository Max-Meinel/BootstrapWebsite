My Projects

Sudoku Generator in Java

I developed a Java program, which creates a number combination as basis a feasible solution within a Sudoku grid!

At first the program puts the numbers 1-9 in every 3x3 square. In case there are duplicates within a line or a column in the overall grid, numbers are swapped within the 3x3 square. This procedure is repeated till no duplicates exist anymore. Although a high number of loops need to be executed the overall runtime is still only some milliseconds. The UI is developed with Java Swing.

Snake Game in Java

I developed the good old snake game in Java (you are a snake in a grid and try to get a very long tail and if you eat an apple, your tail gets longer)!

The program creates an ArrayList for the tail of the snake. In the ArrayList there are tail objects which have an x-axis and a y-axis to position in the grid. If the snake eats the apple, the ArrayList gets one more item added and your score increases. If the snake is colliding itself or the border the game is over. The UI is made with Java Swing.
